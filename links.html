Joe's Awesome Link Page

Check out these links to some cool sites and resources for web developement:

MAIN PAGE PROJECTS LINKS ABOUT JOE

Github.com - Backup and share your projects

If you're working on a webpage or program, you'll want to be able to back up your work. Github does that and a lot more; you can even host a blog at Github Pages. If you are working as part of a team this is an absolute must.

CodeWars.com - Level up your coding skills

CodeWars presents bite-sized coding challanges to sharpen your skills. When you complete a "Kata" you get to compare your sollution to other coders in the community to learn clever new methods and best practices.

Mozilla MDN - Beginning to advanced web developement resource

This is a great place to start learning about HTML, CSS, Javascript, and much more. Learning is arranged into progressively deeper interactive modules that take you step by step into more advanced concepts.

Eloquent Javascript - A great intro to Javascript

Marijn Haverbeke explains starts by explaining the most basic aspects of Javascript and continues on to more advanced concepts. A great companion to MDN, as it breaks down the language into its most basic parts.

W3schools.com - World's largest web developer site

In some ways comparable to MDN. Another great resource when looking for a particular function or method to solve a problem.
